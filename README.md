# Airline-Reservation-System
The Airline Reservation System is a simple console-based application developed in C++. It demonstrates the use of structures, vectors, functions, and pass-by-reference to manage flight information and seat reservations.The program allows users to view available flights and book seats through a menu-driven interface.

## Features
Display available flight details
Book a seat on a selected flight
Prevent bookings when a flight is full
Validate user input for flight selection
Simple menu-driven interface

## Technologies Used
C++
Standard Template Library (STL)
vector
string
Console-based user interface

## Project Structure
Flight Structure
Stores flight number, origin, destination, seating capacity, and booked seats.
displayFlightDetails()
Displays the details of a selected flight.
bookSeat()
Books a seat if seats are available.
Updates the number of booked seats.
main()
Creates sample flight records.
Displays the main menu.
Handles user interaction.
